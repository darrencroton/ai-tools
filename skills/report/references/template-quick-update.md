# Quick Update Template

Use for short status snapshots where a full report would be unnecessarily heavy.

## Required Additional Sections

- `Current State`
- `What Changed`
- `Immediate Next Step`
- `Blockers`

## Completion Gate

Do not finish until the report contains:

- the current state
- what changed since the last known state
- the immediate next step
- blockers or an explicit `None`

## Template

```md
## Objective

## Current State

## What Changed

## Blockers

## Immediate Next Step

## Completion Status
```

## Notes

- Keep this mode short.
- Use it only when the user does not need full evidence and analysis sections.
- If the task has become analytical rather than purely status-based, switch to a fuller mode.

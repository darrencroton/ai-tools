# AI Workspace

Shared AI agent configuration and hand-written skills.

## Contents

- `AGENTS.md`: global instructions used across AI coding assistants
- `setup.sh`: setup script for linking shared configuration into local tool directories
- `tools.conf`: tool registration used by the setup script
- `skills/`: reusable skills that are intended to be version controlled here

## Notes

- Generated files, local artefacts, and selected third-party or separately managed skills are excluded via `.gitignore`
- `skills/ai-orchestrator/` is intentionally not tracked here because it is already managed separately
